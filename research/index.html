Initial Research

When we were first brought onto The Peach Project we were shown the work that a previous team had done. The code was a Node.js application, but it was incomplete and written in quite a haphazard way. We felt uncomfortable continuing with this code, because we knew we'd probably take longer refactoring the code-base than actually developing new features. So we decided to take the best parts of the previous project and use them to build our application from scratch.
